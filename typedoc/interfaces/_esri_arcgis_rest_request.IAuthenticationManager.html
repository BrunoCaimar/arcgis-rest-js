Interface IAuthenticationManager

Authentication can be supplied to request via ArcGISIdentityManager, ApplicationCredentialsManager or {@linkcode APIKeyManager}. These classes implement IAuthenticationManager.

const session = new ArcGISIdentityManager({
  username: "jsmith",
  password: "123456",
  // optional
  portal: "https://[yourserver]/arcgis/sharing/rest"
})

request(url, { authentication: session })
